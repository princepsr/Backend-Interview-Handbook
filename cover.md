## What is This?

The Java Backend Interview Handbook is a structured, depth-first reference for engineers preparing for backend SDE2 interviews at top-tier companies. It covers core Java internals, Spring Boot architecture, distributed systems design, and real-world LLD case studies.

All code is written for **Java 17** and **Spring Boot 3.x** — production-grade patterns, not toy examples.
